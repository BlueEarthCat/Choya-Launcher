# Choya Launcher

**Minecraft 1.20.6 Launcher for Choya Server**

![Choya Launcher](https://blueearthcat.github.io/Choya-Launcher/build/icon.png)

## 소개

Choya Launcher는 **Minecraft 1.20.6 버전**을 위한 전용 런처로, Choya 서버 접속에 최적화되어 있습니다.  
사용자는 별도의 복잡한 설정 없이 간편하게 게임 클라이언트를 실행하고 서버에 접속할 수 있습니다.

---

## 주요 기능

- Windows, macOS, Linux 지원 (x64 / ARM64)
- 자동 업데이트 지원 (`electron-updater`)
- 공식 Choya 서버에 최적화된 설정 포함
- Mod 및 라이브러리 자동 관리
- 쉬운 설치 및 제거

---

## 다운로드

최신 Release 버전은 [GitHub Releases](https://github.com/BlueEarthCat/Choya-Launcher/releases)에서 확인할 수 있습니다.

| 운영체제 | 다운로드 |
|----------|----------|
| Windows  | `Choya-Launcher-v1.0.0-windows.exe` |

## 설치 방법

### Windows

1. `Choya-Launcher-*.exe` 다운로드
2. 설치 프로그램 실행
3. 설치 경로 선택 후 설치 완료
4. 런처 실행 후 자동 로그인 및 서버 접속

### macOS

1. `Choya-Launcher-*.dmg` 다운로드
2. 드래그하여 Applications 폴더에 설치
3. 런처 실행

### Linux

1. `Choya-Launcher-*.AppImage` 다운로드
2. 실행 권한 부여:
   ```bash
   chmod +x Choya-Launcher-*.AppImage
